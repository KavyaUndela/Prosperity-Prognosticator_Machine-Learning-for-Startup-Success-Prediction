## 🚀 Getting Started

### For Windows Users (Fastest)
1. Double-click `setup.bat` - Automated setup
2. Double-click `run.bat` - Start application
3. Open browser to `http://127.0.0.1:5000/`

### For macOS/Linux Users
```bash
chmod +x setup.sh && ./setup.sh    # One-time setup
chmod +x run.sh && ./run.sh        # Run application
```

### For Manual Setup
1. Create virtual environment: `python -m venv venv`
2. Activate it: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Unix)
3. Install dependencies: `pip install -r requirements.txt`
4. Run app: `python app.py`

## 🎓 Usage Examples

### Run Locally
```bash
python app.py
# Access: http://127.0.0.1:5000/
```

### Test API with Python
```python
import requests
response = requests.post('http://127.0.0.1:5000/api/predict', json={
    'funding_total_usd': 1500000,
    'funding_rounds': 3,
    'is_CA': 1,
    'is_software': 1,
    'has_VC': 1
})
print(response.json())
```

### Test API with cURL
```bash
curl -X POST http://127.0.0.1:5000/api/predict \
  -H "Content-Type: application/json" \
  -d '{"funding_total_usd": 1500000, ...}'
```
